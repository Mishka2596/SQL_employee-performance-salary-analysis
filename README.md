# Employee Performance & Salary Analysis

## Project Overview

This project analyses employee performance, salaries, departmental trends, and workforce insights using SQL.

The project simulates a real HR analytics environment where business intelligence techniques are used to answer operational and workforce-related business questions.

The goal of the project is to demonstrate SQL querying, data analysis, and reporting skills using realistic HR data.

---

## Business Questions Answered

1. Which departments have the highest salary costs?
2. Which employees have the highest performance ratings?
3. What is the average salary per department?
4. Which employees earn above the company average salary?
5. Which departments have the highest average performance scores?
6. How many employees are in each department?
7. Which employees qualify as high performers?

---

## Dataset Structure

### Employees Table
- EmployeeID
- EmployeeName
- DepartmentID
- Salary
- PerformanceScore
- HireDate

### Departments Table
- DepartmentID
- DepartmentName

---

## Key Findings

- IT had the highest average salary across departments.
- HR had the smallest workforce size.
- Several employees earned significantly above the company salary average.
- High performers were concentrated within the IT and Finance departments.

## SQL Concepts Used

| Concept | Example Usage |
|---|---|
| SELECT | Retrieve employee data |
| WHERE | Filter high performers |
| GROUP BY | Average salary per department |
| HAVING | Departments with avg salary > threshold |
| ORDER BY | Rank highest earners |
| INNER JOIN | Combine employees with departments |
